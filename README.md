# CPSC-335-Project1

## How to use the program
### 1. Open a terminal window from within the project directory
### 2. Run the command "python main.py"
### 3. Follow the menu selections and select an appropiate option:
> #### a. Selecting 1 will launch Algorithm 1: Connecting Pairs of Persons
> #### b. Selecting 2 will launch Algorithm 2: Greedy Approach to Hamiltonian Problem
> #### c. Selecting 3 will exit the script
